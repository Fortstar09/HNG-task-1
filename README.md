# 📝 Todo Card Component

A responsive, interactive **Todo Card UI component** built with pure HTML, CSS, and JavaScript.  
It provides task management features such as editing, priority tracking, due dates, and status updates — all in a clean card layout.

---

## 🚀 Features

- ✅ Mark task as completed
- 📝 Edit task details via modal
- ⚡ Priority levels (Low, Medium, High)
- ⏳ Real-time due date countdown
- ⚠️ Overdue indicator
- 📂 Expand/collapse long descriptions
- 🏷️ Tag display
- 🔄 Status control (Pending, In Progress, Done)
- 🧠 State-driven UI updates
- 📱 Fully responsive design

---

## 🧱 Tech Stack

- **HTML5**
- **CSS3 (Flexbox + Responsive Design)**
- **Vanilla JavaScript (No frameworks)**

---

## 🚀 How to Run Locally

1. Clone this repository:

   ```bash
   git clone https://github.com/Fortstar09/profile-card.git
   ```

2. Open the project folder:

   ```bash
   cd profile-card
   ```

3. Open the index.html file in your browser.

That’s it — your profile card should now be visible locally!
